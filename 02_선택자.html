#CSS 선택자(Selector)

  • CSS에서는 선택자를 이용해 원하는 요소를 선택할 수 있다
  • tag : 해당 태그인 모든 HTML요소를 선택한다
  • .Class : 해당 클래스 속성(attribute)을 가진 모든 HTML요소를 선택한다
  • #id : 해당 ID 속성(attribute)을 가진 모든 HTML요소를 선택한다
